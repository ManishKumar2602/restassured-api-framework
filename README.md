
# 🚀 API Automation Framework (Rest Assured + TestNG)

## 📌 Overview
This project is an API Automation Framework built using Rest Assured, TestNG, and Maven.

---

## 🧰 Tech Stack
- Java
- Rest Assured
- TestNG
- Maven
- Apache POI
- Jackson
- Log4j2
- Extent Reports

---

## ✨ Features
- API Testing (GET, POST, PUT, DELETE)
- Data-driven testing (Excel & JSON)
- Authentication handling
- Logging using Log4j2
- Reporting using Extent Reports
- Retry mechanism
- Parallel execution

---

## 🧱 Project Structure

src/test/java/com/api
- base
- config
- utils
- services
- tests
- pojo

---

## ▶️ How to Run
1. Clone repo
2. Import as Maven project
3. Run testng.xml
